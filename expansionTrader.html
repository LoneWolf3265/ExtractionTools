FAQ ❯

In this section, we will add your items to the trader file. You will find four new elements to generate your trader file:

1. Number generator

The Number Generator is a powerful tool that creates random numbers within a range defined by you. When using the multi-item-generator or single-item-generator, look for the "Auto Generate" button in some input fields. Clicking this button will generate a random number for each classname entry in the "ClassName" field.

Encountering issues? Here are some tips:

- Ensure you've set the minimum and maximum values for the number generator. - Make sure you have already entered a classname in the respective input field.

2. Multi-item-generator

The multi-item-generator allows you to add multiple classnames at once. It's a convenient way to quickly include various items. Keep in mind that some options like SellPricePercent, QuantityPercent, SpawnAttachments, and Variants are not available in this generator. For such options, refer to the single-item-generator.

3. Single-item-generator

With the single-item-generator, you can focus on generating a single classname while having access to all available options. This gives you more flexibility and customization possibilities compared to the multi-item-generator.

4. Deleted Generated Classnames

To manage your created classnames, use the "Deleted Generated Classnames" section. Simply click on a classname entry to remove it from the list.
